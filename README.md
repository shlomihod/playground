# Playground

Interactive demos for learning AI/ML concepts — each running entirely in the browser.

| Demo | Description | Link |
|------|-------------|------|
| **RAG** | Visualize how Retrieval-Augmented Generation embeds, retrieves, and augments a query | [Live Demo](https://shlomihod.github.io/playground/rag/) |
| **ReAct** | Step through the ReAct (Reasoning + Acting) agent loop with animated tool calls | [Live Demo](https://shlomihod.github.io/playground/react/) |
| **Safeguard** | See how an LLM safety classifier evaluates content against a policy | [Live Demo](https://shlomihod.github.io/playground/safeguard/) |

Every demo and the hub share a light/dark theme toggle; the choice is persisted per-origin and also follows your system's `prefers-color-scheme` on first visit.
